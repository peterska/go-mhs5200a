                         3200/5200 communication protocol

I. Overview:

The control instructions generally adopt the command line method, and the communication rate is 57600. The instructions are issued by the PC , the machine parses and executes, and then returns the results to the PC . The ending character of each command is a newline character. In C language, it is represented by \n ( hexadecimal representation is 0x0a) . The different commands are explained below.

Two categories of instructions:

1. s command : function is used to set parameters

The command format is as follows:

Colon ( : ) + S + channel number (1 or 2) + Command

 

The following command formats all take 1- channel operation as an example

 

( 1 ) :s1f command: set the output frequency of CH1 channel

Such as : s1f100000+ newline character This command is to set the output frequency of CH1 channel to      1000Hz

For example : s1f10 + line feed This command is to set the output frequency of CH1 channel to 0.01Hz

 

( 2 ) :s1w command: set the output waveform type of CH1 channel

 

 

 

      :s1w0+ newline

  Set CH1 channel output waveform to sine wave

      :s1w1+ newline

  Set CH1 channel output waveform to square wave

      :s1w2+ newline

  Set CH1 channel output waveform to sine wave

      :s1w3+ newline

  Set the CH1 channel output waveform to rising sawtooth

      :s1w4+ newline

  Set CH1 channel output waveform to falling sawtooth

  :s1w ( 100-115 ) + line break

  Set CH1 channel output waveform 0-15 arbitrary waveform

 

 

( 3 ) :s1a command: set the amplitude of the output waveform of CH1 channel

 

For example : s1a 0500+ line feed: set the amplitude of the CH1 channel output waveform to 5.00V in the 0dB state, and set the CH1 channel output waveform to 0.500V in the -20dB state .

 

( 4 ) :s1d command: set the duty cycle of CH2 channel output waveform

Such as : s1d999+ line feed: set the duty cycle of CH1 channel output waveform to 99.9%  

( 5 ) :s1o command: Set the offset of CH1 channel output waveform

For example : s1o240+ newline character set the offset of the output waveform of CH1 channel to 120%

For example : s1o120+ newline character set the offset of CH1 channel output waveform to  0%

For example : s1o0+ newline character set the offset of CH1 channel output waveform to -120%

( 6 ) :s1p command: set the phase of the output waveform of CH1 channel

   For example : s1p180++ line feed character sets the output waveform phase of CH1 channel to 180 degrees

( 7 ) Attenuation control command

:s1y1+ line break

        CH1 channel attenuation on

:s1y0+ newline

        CH1 channel attenuation off

:s2y1+ line break

        CH2 channel attenuation on

:s2y0+ newline

        CH2 channel attenuation off

 

 

( 8 ) Switch measurement selection function command

        Select to measure negative pulse width

        Select the duty cycle

 

( 9 ) :s1t command: set sweep time

   Such as : s1t100+ newline character set the sweep period to 100S

( 10 ) : s3f command: set the sweep start frequency

   Such as : s3f100000+ newline character set the sweep start frequency to 1KHz

( 11 ) :s4f command: set the end frequency of sweep

   Such as : s4f200000+ newline character, set the sweep stop frequency to 2KHz 

( 12 ) Gate time setting instruction

   :s1g0+ newline

    Set the frequency measurement time to 10 seconds
